# Tirana 地拉那系统

## 简介
- 地拉那，阿尔巴尼亚首都 
- Spring Boot 经典，且后续持续更新
- 基于vue-element-admin的前端构成
- 结合了企业实践经验，可作为模版落地
- 实现了不同用户的动态权限

## 版本

| soft        | version |
|-------------|---------|
| Java        | 17      |
| Maven       | 3.6.3   |
| Spring Boot | 2.7.7   |
| Mysql       | 8       |
| Redis       | 6       |
| knife4j     | 4.0.0   |


## 功能特点

- 统一异常处理，自动异常数据落盘
- 统一响应处理，使用@History，@HistoryRecord 灵活变通
- 数据表变更实现代理记录
- 序列化枚举
- json数据字段支持
- JWT
